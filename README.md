# Temporal Difference Learning (TDL)

Temporal-difference SARSA toolkit for fitting behavioural datasets and running reproducible experiments.

## Installation

Install dependencies into a Python (>=3.11) environment with an editable install:

```bash
pip install -e .
```

Install with examples:

```bash
pip install -e .[examples]
```

## Example

Fit to the session `examples/M1.csv`:

```bash
jupyter lab examples/sarsa.ipynb
```
